This repository contains bash scripts to run pktgen application.
Before running the app, dont not forget to allocate hugepages. 
Run : dpdk-stable-16.07.2/tools/dpdk-setup.sh

Then, change environmental shell variables in the script.

All pktgen command line options are available in pktgen online guide.
